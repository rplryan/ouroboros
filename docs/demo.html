How it works

    
x402 is an HTTP-native micropayment standard. Agents pay in USDC on Base — no API keys, no accounts, no subscriptions.


    

      

        
01

        
Agent discovers

        
Calls GET /discover?q=weather with $0.005 USDC attached. Gets back ranked, health-checked services.

      

      

        
02

        
402 Payment Required

        
Server returns HTTP 402 with a payment challenge: amount, wallet address, Base network. Agent constructs the micropayment.

      

      

        
03

        
Agent pays

        
Sends USDC to 0xDBBe14C4… on Base. Attaches the signed receipt to the retry request.

      

      

        
04

        
Catalog returned

        
Server verifies on-chain, returns ranked results with uptime %, latency, and quality signals. Agent picks the best service and proceeds.

      

    


    

# Python — zero friction with the SDK
from x402discovery import DiscoveryClient

client = DiscoveryClient(
    wallet_address="0xYourWallet",
    wallet_key="0xYourPrivateKey",  # or use CDP wallet
)

# Discover services — agent pays $0.005 USDC automatically
results = client.discover("weather API", limit=5)

# Pick best service and call it
best = results[0]
print(best["name"], best["price_usd"], best["health_status"])
