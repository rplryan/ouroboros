How x402 works
1

Agent calls the API

An autonomous agent sends a normal HTTP GET or POST to any x402-enabled endpoint.

2

402 Payment Required

The server replies with HTTP 402 and a signed payment request: amount, wallet address, network (Base).

3

Agent pays in USDC

The agent sends a micro-payment (e.g. $0.001 USDC) directly on Base mainnet — no key exchange, no OAuth.

4

Access granted

The server verifies the on-chain receipt and returns the data. The whole round-trip takes ~1 second.

Python in 5 lines
import x402discovery # 1. Find a service services = x402discovery.discover("crypto price") api = services[0] # 2. Pay and call — agent handles the 402 automatically result = x402discovery.pay_and_call(api.url, wallet=wallet) print(result.data) # {'BTC': 95421.00, 'ETH': 3200.50, ...}
